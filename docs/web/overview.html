Get more performance out of your Channel module.

How it works

NSM Turbo Channels is an extension that provides developers with more choice and tuning options to get the best performance out of the {exp:channel:entries} tag.

The un-altered Channel Entries method joins the members and channels tables to you database query and includes a lot of data that you may never even use. This un-needed information increases the size of the data packets that are retrieved from the database. Every row that ExpressionEngine needs to process requires more server resources.

Now developers can specify the custom fields that are returned for a Channel Entries tag. This results in a more economical use of the database giving you, the developer, more control over your website's performance.

Features

  • Extended disable parameter options for increased performance tuning
  • New db_ref parameter to track your database calls in your SQL log
  • Specify the custom_fields columns that are retrieved from the database
